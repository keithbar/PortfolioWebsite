What I've Built

PROJECT_01

XM Scrobbler

A web app that bridges satellite radio and Last.fm, letting users scrobble tracks in real time when using physical devices that have no native scrobbling support. A Node.js server polls a public satellite radio API for currently playing tracks, then handles OAuth authentication with the Last.fm API to scrobble it on the user's behalf. Demo deployed live on Oracle Cloud.

Node.js JavaScript Last.fm API OAuth Oracle Cloud

PROJECT_02

Bark Detector

An AI-powered local tool that listens to your microphone and uses Google's YAMNet model to detect dog barking in real time. Inspired by a particularly noisy neighbour, it logs the time and duration of every barking session and surfaces the data through a dashboard of visualisations, showing patterns by time of day, day of week, and month.

Python YAMNet / TensorFlow ML Inference Data Visualization

PROJECT_03

Checkout Calculator

A single-page retirement calculator with a different question at its core: not "when can I retire?" but "what do I need to retire right now?" Users enter details about their life and expenses and get a precise target number. All calculations run client-side, ensuring personal data remains private and fully self-contained. Hosted on GitHub Pages.
